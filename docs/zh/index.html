11 个 SLASH 命令

完整的 skill 生命周期。

/skill-install

从 GitHub URL 安装。自动分类、写 INDEX、画 graph。

/skill-uninstall

反向清理元数据。幂等。

/skill-update

git pull 一个或全部，刷新 portal 索引。

/skill-list

按域分组打印所有已装 skill。

/skill-recommend

为某任务推荐 load_skills=[...] 组合。

/skill-classify

手工把一个 skill 移到另一个域。

/skill-doctor

5 条规则的健康体检。

/skill-graph-sync

Reconcile graph against INDEX. Idempotent.

/portal-start

把 portal 起来（5174 + 5173）。幂等。

/portal-stop

杀 5173 / 5174 端口上的进程。

/portal-status

PID / 上次索引时间 / skill 数量。

底层 portal。上层 agent。
